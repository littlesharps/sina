关于 Sina
====
##### 环境配置
* VitualBox
* Vagrant
* Homestead
* Git
* Yarn
* Composer
* Heroku
* Laravel 5.5
##### 功能简介
* 用户注册登录（会话机制）
* 访问权限（页面、动作、授权策略）
* 邮件发送（账户激活、密码找回、令牌、模型事件监听）
* 文章CURD（模型关联 一对多）
* 粉丝关注（模型关联 多对多、预加载避免N+1查询）
##### 后续优化
* 文章编辑引入编辑器，支持图片、视频文件的上传等，例如tinymc,富文本编辑器
* 支持点赞，评论
* 等等（待续）
